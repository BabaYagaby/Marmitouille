# Marmitouille
-Pour le sujet, nous avons choisi de parodier un célèbre site de recettes de cuisine.
-Le groupe se compose de Tom Beaurain(tom-brn), Artur Devaux(Linkus-star) et Gabriel Chapron--Gardes(BabaYagaby).
-Aucune fonctionnalité particulière mise en place.
-Pour les techniques et les langages, nous avons principalement utilisé HTML, CSS et Bootstrap.
-Il n'y avait pas d'organisation réelle, tout le monde travaillait sur ce qu'il voulait et ajoutait ce qu'il voulait à condition d'en parler aux autres pour éviter les bugs (pull avant de commit).
-Git ne semblait pas fonctionner de temps en temps, par exemple il a fallu modifier un chemin d'accès pour que Git fonctionne à nouveau.
